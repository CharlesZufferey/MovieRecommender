This friendly application has been designed as such:

  • 1) Encourages you, user, to enter some keywords below
  • 2) Transforms these keywords in an URL and follow the link
  • 3) Scrapes the webpage for all images
  • 4) Makes you, user, patiently wait with its trickery while it proceeds
  • 5) Analyses the images via a neural network to determine its type
  • 6) And finally, issues recommendation on the products which need more pictures
